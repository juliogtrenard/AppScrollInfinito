# Titulo del proyecto

AppScrollInfinito

## Descripción

Aplicación para guardar tareas desarrollada con Kotlin en Android Studio. Permite agregar tareas y luego borrarlas una vez realizadas.

## Primeros pasos

### Cómo usar

* Clona el repositorio en algún IDE y ejecuta el programa.
